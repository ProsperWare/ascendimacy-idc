## Como usar (instrução para o agente)

```
1. Acesse este arquivo:
   https://raw.githubusercontent.com/ProsperWare/ascendimacy-idc/main/sessoes/FILA.md

2. Localize sua empresa na tabela abaixo

3. Acesse a URL do briefing (coluna "Briefing atual")

4. Leia o briefing completo e execute a coleta

5. Commite o resultado conforme o PASSO FINAL do briefing

6. Não altere este arquivo — ele é atualizado pelo agente-coordenador após cada coleta
```
