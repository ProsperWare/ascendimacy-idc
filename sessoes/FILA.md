# FILA DE SESSÕES IDC
> Atualizado manualmente a cada ciclo. Cada agente-empresa lê este arquivo para encontrar seu briefing atual.

---

## Como usar (instrução para o agente)

`
1. Acesse este arquivo:
   https://raw.githubusercontent.com/ProsperWare/ascendimacy-idc/master/sessoes/FILA.md

2. Localize sua empresa na tabela abaixo

3. Crie imediatamente um tab group no Chrome com o nome exato da coluna "Tab group"
   - Clique com botão direito na aba atual -> "Adicionar ao grupo" -> "Novo grupo"
   - Digite o nome exato da coluna "Tab group"
   - Todas as abas desta sessão devem ficar dentro deste grupo

4. Acesse a URL do briefing (coluna "Briefing atual") -- dentro do tab group

5. Leia o briefing completo e execute a coleta, abrindo todas as abas necessárias
   dentro do mesmo tab group

6. Commite o resultado conforme o PASSO FINAL do briefing

7. Não altere este arquivo -- ele é atualizado pelo agente-coordenador após cada coleta
 `
