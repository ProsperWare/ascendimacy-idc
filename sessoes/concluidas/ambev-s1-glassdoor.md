## CONTEXTO

Você está coletando dados para o **Índice de Degralização Corporativa (IDC)** — avalia se uma empresa é **ANDAIME** (prospera quando as pessoas crescem) ou **TETO** (precisa que fiquem pequenas).

**Hipótese para a Ambev:** TETO PERFORMÁTICO (P- E± S-) — a cultura Ambev (derivada da AB InBev / 3G Capital) é estruturalmente orientada à extração máxima de performance com remuneração variável contingente. O ownership model (OKRs agressivos + participação atrelada a metas) cria a aparência de andaime, mas a escala real é limitada: promoção rápida nos primeiros anos → platô brusco após gerência média. Alta portabilidade de saída (marca Ambev abre portas no mercado) → mas a saída é frequentemente involuntária (up or out informal).

**Contexto adicional:** Ambev é a maior cervejaria do Brasil (Brahma, Skol, Antarctica, Corona no BR). Controlada pela AB InBev. Reconhecida como "escola de negócios" informal — ex-executivos Ambev são altamente demandados. Mas no chão de fábrica (operadores, fábricas pelo Brasil) o perfil é radicalmente diferente. O Programa de Trainee Ambev é uma das entradas mais disputadas do país — criar expectativa de andaime que o dia a dia pode não sustentar.

### Tabela de keywords para classificação

| Axioma | → TETO | → ANDAIME |
|---|---|---|
| Proporcionalidade | metas impossíveis, variável nunca pago integral, pressão constante | bônus pago, meritocracia real, salário competitivo |
| Escala | "depois de gerente, travou", platô após 3 anos | crescimento acelerado, promoção rápida no início |
| Saída | demissão por performace sem aviso, up-or-out velado | alumni network forte, "marca Ambev" abre portas |
| Diferencial | "só aprendi a vender cerveja", skills narrow | gestão por OKRs transferível, escola de negócios real |
| Consentimento | cultura de medo, reuniões de confronto, "beat" obrigatório | feedback direto mas respeitoso, autonomia real |
| Dissenso | quem questiona metas é marcado, sorriso obrigatório | debate de ideias aceito, liderança escuta |

### Atenção especial — hipóteses a testar

**Evidências que CONFIRMARIAM teto performático:**
- "Metas impossíveis" como tema recorrente — especialmente em trade/vendas
- Saída involuntária (demissão por resultado) apresentada como "cultura de alta performance"
- Escala real só nos primeiros 2–3 anos, depois platô ou saída
- Trabalho de final de semana / feriado naturalizado como parte da cultura

**Evidências que FALSIFICARIAM (buscar ativamente):**
- Reviews de operadores de fábrica — se o modelo se repete fora do escritório
- Carreira real para quem não virou trainee (progressão da base operacional)
- D&I além do discurso: lideranças diversas na base (não só no PDF de ESG)
- Alumni que ficaram 10+ anos e tiveram escala real = andaime de longo prazo possível

**Bifurcações a testar:**
1. Trainee/Corporate vs. Operadores/Distribuidores — fundamental para IDC
2. Vendas (comercial de campo) vs. Marketing/HQ São Paulo
3. Região Norte/Nordeste (fábricas tradicionais) vs. SP (sede)

**Nota especial:** Registre uso das palavras "dono" e "ownership" — são centrais na cultura Ambev. Se usadas positivamente = possível andaime real para quem chega. Se usadas com ironia/amargura = teto disfarçado.
