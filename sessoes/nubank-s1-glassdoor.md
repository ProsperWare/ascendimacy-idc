## CONTEXTO

Você está coletando dados para o **Índice de Degralização Corporativa (IDC)** — avalia se uma empresa é **ANDAIME** (prospera quando as pessoas crescem) ou **TETO** (precisa que fiquem pequenas).

### Tabela de keywords para classificação

| Axioma | → TETO | → ANDAIME |
|---|---|---|
| Proporcionalidade | microgestão, controle, sem autonomia | autonomia, confiança, ownership |
| Escala | sem alternativa, panelinha, favoritismo | múltiplas trilhas, mobilidade |
| Saída | retaliação, medo, não deixam sair | alumni forte, portas abertas |
| Diferencial | estagnação, sem crescimento, teto | mentoria, investimento, aprendizado |
| Consentimento | bait-and-switch, mudaram regras | transparência, previsibilidade |
| Dissenso | quem discorda é cortado, echo chamber | feedback real, escutam |

### Atenção especial (hipótese Gaiola Dourada)

Fique atento a padrões de:
- Salário e benefícios excepcionais usados como argumento para ficar ("mas paga muito bem")
- Stock options com vesting longo que tornam a saída cara
- Pressão implícita para não sair ("você vai perder o vesting")
- Cultura muito forte que torna a saída psicologicamente custosa
- Sentimento de que o mercado externo é "pior" (narrativa cultivada)
